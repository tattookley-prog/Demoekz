# Demoekz

Набор скриптов для Proxmox VE, подготовленный для демоэкзамена.

---

## proxmox_setup.sh

Интерактивный bash-скрипт с меню для управления инфраструктурой Proxmox VE.

### Возможности

| Пункт меню | Описание |
|---|---|
| Информация об узле | Версия PVE, статус ресурсов |
| Список ВМ и контейнеров | `qm list` + `pct list` |
| Создать LXC-контейнер | Интерактивное создание контейнера |
| Создать виртуальную машину | Интерактивное создание KVM-ВМ |
| Управление снапшотами | Создание и просмотр снапшотов |
| Состояние хранилища | `pvesm status` + `df` |
| Сетевые настройки | IP-адреса и мосты |
| Обновить Proxmox | `apt dist-upgrade` |

### Требования

- Proxmox VE 7 или 8
- Запуск **от имени root** на узле PVE

### Запуск

```bash
chmod +x proxmox_setup.sh
bash proxmox_setup.sh
```

> Скрипт будет расширяться в процессе подготовки к демоэкзамену.
